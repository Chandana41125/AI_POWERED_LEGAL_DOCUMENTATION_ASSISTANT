# 🚀 LexiAI - AI Powered Legal Document Assistant

## 📌 Overview
LexiAI is an AI-powered web application that helps users analyze legal documents efficiently. It allows users to upload files, generate summaries, translate content, and interact with an AI chatbot.

---

## ✨ Features
- 🔐 User Authentication (Login / Signup)
- 📄 Upload PDF, DOCX, TXT files
- 🤖 AI-based document summarization
- 🌐 Multi-language translation
- 💬 Chatbot using Ollama (Llama3)
- 📥 Download summary as PDF / DOCX
- 📁 Draft document management

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Database:** MySQL
- **AI Model:** Ollama (Llama3)

## ▶️ How to Run the Project

1. Clone the repository:

git clone https://github.com/your-username/AI_POWERED_LEGAL_DOCUMENTATION_ASSISTANT.git


2. Navigate to project folder:

cd AI_POWERED_LEGAL_DOCUMENTATION_ASSISTANT


3. Install dependencies:

pip install -r requirements.txt


4. Configure environment:
- Add your `config.py` or `.env` file with database credentials

5. Run the application:

python app.py


---

## ⚠️ Important Notes
- Sensitive files like `.env`, `config.py`, and `venv/` are not included for security reasons
- Ensure Ollama is running locally for chatbot and summarization features
